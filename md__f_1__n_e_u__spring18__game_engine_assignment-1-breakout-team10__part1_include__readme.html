Information about include directory

This folder contains all header files needed for Breakout Game.
GameObject is base class for Ball, MainMenu, Brick, and Paddle.
LTexture is responsible for handling texture.
LTimer is responsible for capping frames per second.
ManagerHeader contains SDLGraphicsProgram which is responsible for managing game.
ResourceManager is responsible for caching assests.
